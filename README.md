# TICC_economic
applying TICC to understand the economy of a country

In this [demo video](https://youtu.be/7iuquZBQ_No), I try applying Toeplitz Inverse Covariance-based for Clustering to cluster seven US economic indicators. The multivariate dataset is a monthly data that spans from 2001-2025.
To run the code in the notebook `economicUS.ipynb`, you need to download the data first. The source of the data is at the end of the notebook.
